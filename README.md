## Come compilare con Cmake
>
> ~/DomoticSystem/$ mkdir build
>
> ~/DomoticSystem/$ cd build
>
> ~/DomoticSystem/build/$ cmake ..
>
> ~/DomoticSystem/build/$ make
>
> ~/DomoticSystem/build/$ ./main
